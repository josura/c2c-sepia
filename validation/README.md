## Methodology 
1. Compare the results of the algorithm with some data taken at two different timesteps, the perturbation values obtained from the algorithm after a time should have the same behaviour as the real data
2. take three different systems, one ill drugged, one ill not drugged and one normal, and see the convergence values of the drugged and normal, if they differ the drug has not worked, if they do not differ the drug has worked
